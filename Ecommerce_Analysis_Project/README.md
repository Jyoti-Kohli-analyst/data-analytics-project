Ecommerce analysis project
This project analyzes ecommerce transaction data using SQL and Python

Dataset:
data.csv

Questions solved using SQL:

1. Unique customerID
2. Unique invoice no.
3. Total revenue
4. Top customers
5. Top selling products
6. Country with highest sales
7. Monthly sales trend
8. Average order value
9. Most expensive products

Questions solved using Python:

1. Monthly sales visualization
2. Top countries chart
3. Top products chart
4. Orders per month chart

Tools used :
SQL 
Python
Pandas
Matplotlib
Seaborn

Output:
SQL queries for analysis
Python notebook for visualization
